## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* python libraries
  ```sh
  bash ./dependencies.sh
  ```

### Installation

Below is an example of how you can set up your led strip.

1. Clone the repo
   git clone https://github.com/seaver-olson/MonitorBacklight
   
2. Upload `main2.ino` onto your arduino
   
3. setup led strip; `DATA_PIN` is defaulted on 8

4. run dependencies.sh

5. run `ordinator.py` on your pc
